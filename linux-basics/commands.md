# Linux Basics

##User Commands

Commands basics

- ~ pasta home

- $ usuário comum
	
- Sudo su para trocar pra usuário root
	
-	# usuário root
	
- exit ou cntl + d para voltar ao usuário normal ou encerrar a sessão
	
- ls • listagem
	
- man + comando manual

- —help • manual resumido
	
- sudo • permissões (super user do)
	
- clear  ou cntrl + l • limpar terminal
	
- q quit
	
-	mkdir (make directory)•  + nome da pasta 
- para criar mais de uma pasta separar com espaço, para criar com nome composto  usar aspas

-	cd (change directory) • + nome do diretório+/
- Para completar os comandos basta pressionar tab
	
-	cd ~ para voltar pra home 
	
-	pwd • imprime o caminho do diretório, endereço do arquivo
	
-	whoami  • mostra nome do usuário
- >>  • pega a informação do comando anterior e joga para um arquivo de sua preferência 
	
- Para acessar uma pasta de nome composto deve se usar \ ex: curso\ de\ terminal/ ou utilizar aspas

-	touch • criar arquivos 

-	nano • editar arquivos  - grande bloco de notas	
	
- - ^ = control
	
-	- Crntl x para sair 
	
-	cat • mostra o conteúdo do arquivo
	
-	mv • modificar o nome dos arquivos
	
-	cp • copiar (nome do arquivo endereço ~ /nome do diretório e /nome da pasta /

-	cd .. • Volta pasta/ diretório
	
-	find • .(a partir deste diretório) -name (nome do arquivo).   Para procurar por uma parte do nome usar oq vc quer
	
-	Para arquivos muito grandes usar head (primeiras linhas) ou tail (últimas linhas) 

-	Utilizar o comando less para ver todo o arquivo mas o carregar em partes
	
-	rm para remover um arquivo 
	
-	rmdir para remover diretório
	
-	rm -i apgar aceitando todas as mensagens

-	Para apagar um diretório com arquivos dentro deve se usar rm -rf ’nome do diretório’ 

-	hostname saber o nome da sua máquina
	
-	hostname -I mostra o ip

- useradd 
	
-	passwd + nome do ususario + senha

- echo - permite que o usuário escreva strings de caracteres para a saída padrão

- unzip - descompactar arquivos
	
-	bash - principal função controlar sistemas operacionais
	
-	df -h - mostra o armazenamento disponível e usado.

- du * -hs lista os arquivos mostrando o tamanho de cada um
	
-	-Y aceita todas as perguntas, até mesmo remoção.
	
-	shutdown now - desligar maquina
	
-	cat /etc/redhat-release - verificar a verção do CentOS7

- cat /etc/os-release - verificar qual o sistema operacional
	
-	cat /proc/cpuinfo - infromações de harware do pc

- uptime - verificar quanto tempo aquele dispositivo está ligado

- lspci - verificar informações de hardware

-lscpu - verificar informações sobre o processador

- editores de texto (vim, vi, nano...)
